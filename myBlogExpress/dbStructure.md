# 数据库结构

## 1. Admin 

1. 用户名：userId
2. 用户密码：userPwd

## 2. Blog
1. Blog标题: blogTitle
2. Blog内容: blogCont
3. Blog发布时间: blogCreateTime
4. BlogId : 博客的id(作为标识)

## 3. Comment

1. comment评论内容: cmtMsg
2. comment评论时间: cmtTime
3. 留言人昵称: name
4. 留言人email: email

## 4. Tags
1. 标签 : tag
2. Blog访问量: visits
3. BlogId : 博客的id(作为标识) 

## 5. EveryDay

1. 每日一句日期
2. 每日一句内容
3. 每日一句作者
4. 每日一句内容翻译
